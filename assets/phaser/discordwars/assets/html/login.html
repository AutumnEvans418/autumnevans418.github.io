Why do you need my bot token?

Because it costs money to run a server with a bot and it's dangerous to store a bot token on the client, it made more sense for the bot token to be entered by the user. The token is cached only for the session.

Fortunately, no permissions are needed for this game to work, and it's easy to create a bot. To create a bot, you need to created a discord application Here

After creating an application, you can get the bot token by going to the "Bot" tab and clicking on the "Copy" button.

To add the bot to a server, use this url with the client id from the "General Information" page of the discord application page: https://discordapp.com/oauth2/authorize?client_id=CLIENT_ID_HERE&scope=bot
