How to Play

Discord Wars is played using discord chat and a person streaming the game. There are a few commands used to control the game

Every command can be typed with just the first letter, like '!j 1' instead of '!join 1'

  • !join 1 = joins team 1. !join red or !j r can also be used
  • !leave = leaves the game. Can also do !l
  • !move 1 2 = moves units from base 1 to 2. Can also do !m 1 2
  • !move 1 = moves all units to base 1.
  • !upgrade 1 = upgrade/heals base 1. Can also do !u 1
  • !say hello! = puts a message in chat. Can also do !s hello!
  • !retreat 1 = sends all units away from base 1. Can also do !r 1

Local Play

To play without discord chat, you can use the command line provided at the bottom of the screen
